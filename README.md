# Geak Audio 果壳智能音响播放工具

请先安装最新版 node js

然后 npm ci

## 播放云听电台直播或昨日回放

`node radio-cn.js`

根据提示选择频道即可播放

## 播放器操作

`node action.js 操作`

### 播放

`node action.js play`

### 停止

`node action.js stop`

### 暂停

`node action.js pause`

### 上一首

`node action.js previous`

### 下一首

`node action.js next`

### 播放器状态

`node action.js status`

### 当前播放信息

`node action.js info`

### 设置音量

`node action.js volume 音量值`

音量值范围 0-100
